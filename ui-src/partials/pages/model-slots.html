模型命名
保存命名规则,客户端看到的 model ID 会实时套用规则(不改原始 ID,不会叠加)。旧 ID 请求也仍然可用。
保存规则后立即生效:客户端请求里的 model 字段将看到改名后的 ID,旧 ID 也仍然可用。 改规则不会叠加——规则是全局模板,改一次就替换前一次。
高级: 自定义完整模板
展开高级模式后,前缀/后缀 input 失效,新 ID 完全由模板决定。可用字段: {prefix} {model} {provider} {suffix}。
