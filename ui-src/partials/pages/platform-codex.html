开启后 Codex 通过 AnyBridge 本地代理转发请求（127.0.0.1:7450），适合供应商只支持 Chat Completions 或需要协议转换的场景。关闭后 Codex 直接连接供应商的 Responses 端点。

开启后 AnyBridge 向 Codex 桌面版注入解锁脚本，让所有模型在模型选择器里显示。关闭后 Codex 桌面版只能切换到默认模型，CLI 不受影响。

开启后 Codex 桌面版使用 auth.json 中的官方 OAuth 登录态认证，模型选择器显示官方模型列表，无需 CDP 注入。请求仍经本地代理转发到第三方 API。仅适用于提供官方模型名的第三方渠道商。

开启后切回官方配置时保留 model_provider 标签，官方与第三方会话历史统一在同一个桶中显示。关闭后切回官方时会话历史回到默认 openai 桶。
